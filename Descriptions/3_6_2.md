### 🛠️ Step 2: Idler Pulley Installation

In this step, we will install the idler pulleys.

### 📝 Instructions
1. Position idler pulleys in designated locations
2. Install mounting brackets and hardware
3. Align pulleys with belt path
4. Secure all fasteners with appropriate torque
