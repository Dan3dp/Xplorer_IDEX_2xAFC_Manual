### 🛠️ Step 4: Install final aluminium profile

In this step, we will install the final aluminium profile to complete the frame structure. This is the last step in the aluminium profile installation process.

### 📝 Instructions
1. **Flip the frame on one side**.
2. Slide the 580mm long aluminium profile in its designated position as shown.
3. Take 2x SHCS M8x35 screws, apply a drop of Loctite 243 to threads, and tighten aluminium profile from both sides as shown.
4. **<span style="font-size: 1.1em; color: #c60808;">Ensure that 580mm aluminium profile does not rotate while tightening screws</span>**

### ℹ️ **INFO**
**<span style="font-size: 1.1em;">Side profile installation completes the structural frame and ensures proper alignment for all components.</span>**
