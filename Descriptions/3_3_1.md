### 🛠️ Step 1: Y-Axis Motors Installation

In this step, we will install the GT2 Pulley on the shaft of the Y-axis motors. 

#### 📦 Bill of Materials (BOM)

| **Qty** | **Part Image/Icon** | **Component Description** |
|:---:|:---:|:---:|
| **2** | ⚙️ | **Fixation plates for Y-Axis Motors** |
| **2** | ⚙️ | **Nema17 48mm Steppers** |
| **2** | ⚙️ | **Antivibration Gasket** |
| **2** | ⚙️ | **GT2 20T 9mm Pulley** |
| **8** | 🔩 | **FHCS M3x10 screw** |

---

### 📝 Instructions

1. Attach a 20-tooth GT2 pulley for a 9mm wide belt to each Nema17 stepper
motor
2. There must be a 6mm gap between the GT2 pulley and the stepper flange.

**<span style="font-size: 1.1em; color: #ff6b6b;">⚠️ IMPORTANT NOTE:</span>** <span style="font-size: 1.1em; color: #ff6b6b;">This step needs to be completed **twice** to create two identical Y-motor assemblies.</span>
