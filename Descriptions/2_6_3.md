### 🛠️ Step 3: Install aluminium profiles

In this step, we will install the upper and lower aluminium profiles to complete the frame structure.

### 📝 Instructions
1. Take 4x BHCS M8x16 screws, apply a drop of Loctite 243 to thread, and preassemble them at the ends of 580mm aluminium profiles
2. Leave between screw head and aluminium profile face approx. **3mm space**
3. Slide upper aluminium profile frames into position as indicated.
4. Using a 5mm Allen key, access screws head through passthrough holes in pillars and tighten 4x BHCS M8x16 screws
5. **<span style="font-size: 1.1em; color: #c60808;">Ensure that 580mm aluminium profiles do not rotate while tightening screws</span>**
6. Proceed same way with aluminium profile from bottom
7. **<span style="font-size: 1.1em; color: #c60808;">Make sure orientation of aluminium profile from printer bottom is correct so that its hole is aligned with hole in profile on back side</span>**

### ℹ️ **INFO**
**<span style="font-size: 1.1em;">Proper profile installation ensures frame stability and correct positioning for subsequent assembly steps.</span>**
